Scops (декабрь 2021)

Сайт для хранения наблюдений за птицами.

Позволяет создавать аккаунты, добавлять записи о наблюдениях, изменять настройки аккаунта и просматривать записи других пользователей.

Написан на PHP, JavaScript и MySQL в качестве учебного проекта по базам данных и интернет-программированию.

Посмотреть можно здесь: Scops

nullproject (сентябрь-декабрь 2021)

Большой проект, над которым я работала в команде из 6 человек в течение семестра.

Суть проекта заключалась в создании личного кабинета пациента для удобной записи на прием к врачу через сайт IGIS. Мы использовали ASP.NET Core, SQL Server и Entity Framework для создания прототипа личного кабинета и моделирования процесса занятия номерков. Заодно мы предложили обновленный дизайн системы.

Mоя основная работа заключалась в сборе и анализе требований от заказчика и проектировании основных модулей системы. Также я занималась реализацией процесса занятия номерков и мелкими правками верстки сайта.

VisualSorting (июль-август 2021)

Небольшая визуализация 5 алгоритмов сортировки: пузырьком, вставками, выбором, сортировка кучей и быстрая сортировка.

Позволяет сортировать от 2 до 200 чисел и замеряет время, потраченное на сортировку. Протестировать можно здесь: jackeeet.github.io/VisualSorting

Все работает на HTML и CSS с чистым JavaScript.
Исходный код здесь: github.com/Jackeeet/VisualSorting

GSS (июнь 2021)

Практическая работа за второй курс - автоматизированная система управления складом.

Десктопное приложение, формирующее автоматические заказы на поставки на основании истории продаж гипотетического продуктового магазина.
Есть возможность просматривать содержимое склада, историю поставок и продаж, а также добавлять записи о продажах и поставках отдельных продуктов.

Над этим проектом мы работали в команде. Мы использовали C# и SQL Server, параллельно изучая Entity Framework.
Моей основной задачей было создание алгоритма, определяющего состав заказа на основе сроков годности отдельных продуктов, их условий хранения, количества на складе и истории продаж.

Исходный код проекта здесь: github.com/DeGrimer/GSS

boids (апрель-май 2021)

Двумерная реализация боидов - модели формирования стай.

Можно менять количество боидов на экране и задавать такие параметры, как скорость перемещения, радиус видимости отдельного боида и минимальное расстояние, на которое один боид готов подпускать других.

Исходники на Python здесь: github.com/Jackeeet/boids

Martius (май 2021)

Курсовая работа по ООП - система хранения и обработки данных о недвижимости, сдаваемой в аренду.

Позволяет создавать, хранить и редактировать информацию о помещениях, людях, арендующих эти помещения, и детали договоров об аренде.
Списки договоров и помещений фильтруются по нескольким параметрам: статус договора об аренде, количество комнат в помещении, наличие мебели и т. п. Для списка арендаторов реализован поиск по ФИО, номеру телефона или паспорта. Также все списки поддерживают сортировку по соответствующим параметрам.

Если в базе данных есть записи об аренде одного и того же помещения одним и тем же арендатором несколько раз подряд, то при создании нового договора программа автоматически рассчитывает скидку.
Процент скидки, необходимое количество договоров, идущих подряд, и другие параметры могут быть изменены в настройках.

Программа написана на C#, в качестве СУБД используется SQL Server. Исходный код здесь: github.com/Jackeeet/Martius
