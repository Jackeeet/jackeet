GSS (June 2021)

An automated warehouse management system created as a second-year summer practice project at my university.

It’s a desktop application for an imaginary grocery shop. It creates a list of items to restock based on the shop’s sales history. Besides that, the app allows the user to view the list of items in stock, sales history, and supplies history. The user can also manually enter data about new sales or supplies.

This was a small team project. We used C# and SQL Server, learning Entity Framework on the go.
My specific job was to create an algorithm that forms the list of goods to restock. The goods are selected based on their shelf life, storage requirements, sales history, and number of items in stock.

You can find the source code here: github.com/DeGrimer/GSS

Martius (May 2021)

A course project for the Object-oriented Programing course at my university.

It’s a desktop application for managing data about rental property lease agreements. The user can create, view, and edit information about real estate property, tenants, and lease agreement details.
Property and lease agreement data can be filtered by parameters such as agreement status, furnishing, or number of rooms. The user can also search for specific tenants by their passport number, their phone number or simply by name.

The system can automatically calculate and apply discounts when a new lease agreement is created. Parameters like discount percentages can be changed in the app’s settings.

I built this application using C# and SQL Server. Its source code can be found here: github.com/Jackeeet/Martius
